What is the OWB Service?

The Open Word Bible (OWB) Service is a self-contained web application designed to help users manage, annotate, and analyze digital books. It includes tools for tagging named entities, customizing layout, managing notes, and visualizing word-level data.

How to Run the OWB Service
  1. Unzip the downloaded owb_service.zip file.
  2. Open a terminal and navigate into the extracted folder:
    3. cd owb_service
  3. Create a python virtual environment
    4. python -m venv env
  4. Start the python Environment (Windows)
    5. env\Scripts\activate
  5. Start the python Environment (linux)
    6. source env/bin/activate
  6. Install the required dependencies (ensure you have Python 3 installed):
    7. pip install -r requirements.txt
  7. Run the service using the following command:
    8. python owb_service.py
